# CS2_SpeedometerHUD_WASD
Плагин CS2 SpeedometerHUD+WASD представляет собой продвинутый 3D-спидометр, который отображает Текущую скорость игрока в единицах игры и Нажатые клавиши движения в формате WASD + дополнительные клавиши

## Краткая суть плагина
Этот плагин показывает вашу текущую скорость в CS2 вместе с визуализацией нажатых клавиш движения (WASD, прыжок, приседание и другие). Он отображается как 3D-интерфейс в игровом мире, а не на экране, что создает эффект "встроенного в игру" спидометра. Особенно полезен для тренировок движения, изучения механик бхопа и страфов.

### ОСНОВНЫЕ ВОЗМОЖНОСТИ
~~~
Отображение текущей скорости - показывает скорость движения игрока в единицах игры
Визуализация клавиш движения - показывает, какие клавиши (WASD, Shift, Space, E, R, F, Tab) нажаты в данный момент
Режим наблюдателя - автоматически показывает скорость игрока, за которым вы наблюдаете
Стабильное отображение - использует метод point_orient для устранения тряски текста
Гибкая настройка - можно менять цвет, позицию, шрифт, размер и другие параметры
Отслеживание прыжков - показывает "JU" при прыжках, включая распрыжки
Работа в реальном времени - обновляется каждые 0.01 секунды для плавного отображения
Индивидуальное управление - каждый игрок может включать/выключать спидометр для себя
~~~
### КОНФИГУРАЦИОННЫЕ ПАРАМЕТРЫ
~~~
Основные настройки
css_speedometer_enabled true/false (true) - Включить или выключить весь плагин
css_speedometer_hud_channel 0-255 (1) - Номер канала для отображения HUD (чтобы не конфликтовать с другими плагинами)
css_speedometer_update_interval 0.01-1.0 (0.01) - Как часто обновлять скорость (в секундах)

Позиция на экране
css_speedometer_hud_x -100.0 до 100.0 (-2.3) - Горизонтальное положение (отрицательные значения - левее)
css_speedometer_hud_y -100.0 до 100.0 (-3.5) - Вертикальное положение (отрицательные значения - ниже)
css_speedometer_hud_z 0.0 до 20.0 (6.7) - Высота от игрока (чем больше, тем выше над головой)

Внешний вид текста
css_speedometer_font_size 10-100 (40) - Размер шрифта в пикселях
css_speedometer_font_name строка ("Consolas") - Какой шрифт использовать (должен быть установлен в системе)
css_speedometer_units_per_px 0.001-1.0 (0.0057) - Масштаб текста (меньше = мельче)
css_speedometer_color строка ("255,255,255") - Цвет текста в формате "красный,зеленый,синий"
css_speedometer_use_bold_font true/false (true) - Использовать жирное начертание шрифта
css_speedometer_text_border_width 0.0-10.0 (0.0) - Толщина обводки вокруг текста
css_speedometer_text_border_height 0.0-10.0 (0.0) - Высота обводки вокруг текста

Функциональные настройки
css_speedometer_show_keys true/false (true) - Показывать клавиши движения или только скорость
css_speedometer_show_spectator true/false (true) - Показывать спидометр когда мертв или в режиме зрителя
css_speedometer_hud_method 0-1 (1) - Способ отображения: 1=стабильный, 0=быстрый (может трястись)

Настройки прыжков
css_speedometer_jump_display_time 0.1-1.0 (0.4) - Сколько секунд показывать "JU" после прыжка
css_speedometer_bunnyhop_window 0.1-1.0 (0.5) - Время для распрыжки (следующий прыжок в этом окне считается распрыжкой)

Логирование
css_speedometer_log_level 0-2 (0) - Детальность логов: 0=только ошибки, 1=ошибки+инфо, 2=все сообщения
~~~

### КОНСОЛЬНЫЕ КОМАНДЫ
~~~
Основные команды
css_speedometer_help - Показать всю справку по командам
css_speedometer_settings - Показать текущие настройки плагина
css_speedometer_toggle - Включить/выключить спидометр лично для себя
css_speedometer_test - Проверить работу плагина (покажет тестовое сообщение)
css_speedometer_reload - Перезагрузить настройки из файла
css_speedometer_enable - Включить плагин для всех
css_speedometer_disable - Выключить плагин для всех

Команды настройки (примеры)
css_speedometer_setposition -2.3 -3.5 6.7 - Изменить позицию спидометра
css_speedometer_setcolor 255,0,0 - Сделать текст красным
css_speedometer_setfontsize 35 - Уменьшить размер текста
css_speedometer_setmethod 1 - Включить стабильный режим (рекомендуется)
css_speedometer_setjumptime 0.6 - Увеличить время показа прыжка

Быстрые переключатели
css_speedometer_togglekeys - Быстро вкл/выкл показ клавиш
css_speedometer_togglespectator - Быстро вкл/выкл показ в режиме зрителя
css_speedometer_togglebold - Быстро вкл/выкл жирный шрифт

ПРИМЕРЫ РАБОТЫ ПЛАГИНА
Как это выглядит в игре:

       285
TA     W  E  R
SH  A  S  D  F
DU     JU

Пример 2: Игрок в режиме зрителя смотрит на другого игрока

[PlayerName]
       310
TA     W  E  R
SH  A  S  D  F
DU     JU

Пример 3: Только скорость (без клавиш)
250

Расшифровка отображаемых клавиш:
TA - Tab (таблица лидеров)
W/A/S/D - Клавиши движения
SH - Shift (приседание/тихая ходьба)
E - Использование (подобрать, открыть)
R - Перезарядка
F - Осмотр оружия
DU - Ctrl (приседание)
JU - Space (прыжок) или колесо мыши вниз
~~~

### ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ
~~~
Требования для работы:
CounterStrikeSharp версии 1.0.362 или выше
.NET 8 runtime
Плагин GameHUD API (обязательно, без него плагин не работает)


Особенности работы:
Не показывает спидометр ботам и HLTV
Автоматически скрывает/показывает при смерти/возрождении
Сохраняет настройки каждого игрока отдельно
Поддерживает горячую перезагрузку сервера плагинов
~~~

### УСТАНОВКА И НАСТРОЙКА
~~~
Простая установка:
Убедитесь, что установлен CounterStrikeSharp последней версии
Установите плагин GameHUD API
Поместите файл CS2_SpeedometerHUD_WASD.dll в папку plugins
Перезагрузите сервер или плагины
~~~
